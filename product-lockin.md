# Project Title
Goalscope

## Problem Statement
Football fans have to visit many different websites to find player information, team standings, and match schedules from European football leagues.

## Proposed Solution
A single website where fans can search for any player, see all league rankings, check upcoming matches, and find award winners from five major European football leagues in one place..

## Key Features
- Search and find any football player's statistics
- View league rankings and standings
- Check upcoming matches and schedules
- See top scorers and award winners

## Technical Approach
- Frontend: Streamlit
- Backend: Python + SQLite
- Database: SQLite
- Data Source: API-Football.com (RapidAPI)

## Success Metrics
- All features are working
- Response Time < 3 sec
